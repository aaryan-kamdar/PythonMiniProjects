# Slot Machine Simulator

A terminal-based slot machine simulator built with Python. Players can deposit money, place bets, spin the reels, and win payouts based on matching symbols.

## Features

* Deposit and balance management
* Betting on multiple lines
* Randomized slot machine spins
* Winning line detection
* Dynamic payout calculation

## Technologies Used

* Python 3
* Random

## How to Run

```bash
python SlotMachine.py
```

## How to Play

1. Deposit an initial amount.
2. Choose the number of lines to bet on (1–3).
3. Place a bet on each line.
4. Spin the slot machine.
5. Win payouts based on matching symbols.

## Symbol Values

| Symbol | Value |
| ------ | ----: |
| A      |    5x |
| B      |    4x |
| C      |    3x |
| D      |    2x |
