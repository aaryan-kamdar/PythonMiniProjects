# Timed Math Quiz

A Python-based math quiz game that generates random arithmetic problems and measures how quickly users can solve them.

## Features

* Random math problem generation
* Addition, subtraction, and multiplication
* Time tracking
* Multiple questions per session
* Score evaluation

## Technologies Used

* Python 3
* NumPy
* Time

## How to Run

1. Install NumPy:

```bash
pip install numpy
```

2. Run the application:

```bash
python TimedMath.py
```

## Gameplay

* Solve 10 randomly generated math problems.
* Continue until all questions are answered correctly.
* The total completion time is displayed at the end.
